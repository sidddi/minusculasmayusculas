2. Objeto y actividad del sitio web

El sitio web minusculasmayusculas.com ofrece una herramienta online gratuita para la conversión de texto entre distintos formatos tipográficos: mayúsculas, minúsculas, capitalización por palabra y mayúsculas tras punto. La herramienta es de uso libre, sin registro ni coste alguno para el usuario.

El procesamiento del texto introducido por el usuario se realiza íntegramente en el navegador del propio usuario (lado cliente). No se envía ningún dato de texto a ningún servidor externo ni propiedad de este sitio web.
